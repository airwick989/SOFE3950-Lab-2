1. cd
'cd <directory>' changes the current working directory to <directory>. If the <directory> argument is not present, it will report the current directory
2. clr
'clr' will clear the screen
3. dir
'dir <directory> lists the contents of directory <directory>
4. environ
'environ' lists all the environment strings
5. echo
'echo <comment>' displays <comment> on the terminal
6. help
'help' will display the user manual on the terminal
7. pause
'pause' will pause the operation of the shell until the Enter key is pressed
8. quit
'quit' will exit the shell
9. <programname>
Entering anything other than the above commands will be interpreted as a program invocation of the program named <programname>
